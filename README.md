# React-Shadcn-React-Router-Dom-template

This is a template for Create React App configured with Craco to get `@/ alias`` and React Router Dom.

# Getting Started

1. Clone this repo
2. Run `pnpm install`
3. Then start the development server with `pnpm dev`

# Features

- [x] React Router Dom
- [x] Craco for `@/` alias
- [x] Shadcn
- [x] Pnpm add
